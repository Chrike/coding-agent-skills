# Archive Boundary

No assets are currently archived.

This directory is a non-runtime governance boundary. It is never installed by a profile and does not participate in routing.

An asset may enter Archive only after decision-grade evidence shows that it has no retained safety, routing, or maintenance value, with an explicit reason, source/revision references, rollback or recovery condition, and maintainer approval. The lifecycle policy does not automate that action, and this file does not declare any current asset low-value.

Git history and the asset lifecycle ledger preserve current and historical source context without copying runtime rule bodies here.
